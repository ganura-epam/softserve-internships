# Self-Assessment Questions for SoftServe Internship Tracks

## Instructions for Google Forms

### How to Create Google Forms:
1. Go to: https://forms.google.com/
2. Click **+ Blank** or use a template
3. Add title: "SoftServe [Track Name] - Self Assessment"
4. Add description explaining the purpose
5. Copy questions from sections below
6. Use question types: Multiple choice, Checkboxes, Linear scale, Short answer
7. Enable "Response validation" for required fields
8. Click **Settings** → Enable "Collect email addresses"
9. Click **Send** → Get shareable link

## Scoring Guidelines (For Internal Use)

### Scoring System:
- **Beginner (0-30 points)**: Limited experience, strong learning motivation required
- **Intermediate (31-60 points)**: Some hands-on experience, good foundation
- **Advanced (61-80 points)**: Solid practical experience, ready for internship
- **Expert (81-100 points)**: Extensive experience, strong candidate

### Point Allocation:
- Linear scale 1-5 questions: 1pt, 2pt, 3pt, 4pt, 5pt
- Multiple choice experience: 0pt (none) to 5pt (expert)
- Checkboxes: 1pt per selection (max 5pts per question)

## Google Forms Setup Instructions

### Step 1: Create Forms (15 minutes each track)

1. **Go to Google Forms**: https://forms.google.com/
2. **Create new form**: Click **+ Blank**
3. **Add title**: "SoftServe [Track Name] Self-Assessment"
4. **Add description**:
   ```
   This self-assessment helps us understand your current skill level for the [Track Name] internship.
   Please answer honestly - this is not a test, but a way to tailor the internship to your needs.
   ```

### Step 2: Add Sections

Create 2 sections:
1. **Section 1**: Basic Information (Questions 1-5)
2. **Section 2**: [Track Name] Skills (Questions 6+)

### Step 3: Configure Settings

1. Click **Settings** (gear icon)
2. **General**:
   - ✅ Collect email addresses
   - ✅ Limit to 1 response
   - ✅ Respondents can edit after submit
3. **Presentation**:
   - ✅ Show progress bar
   - Confirmation message: "Thank you! We'll review your assessment and contact you soon."

### Step 4: Add Response Validation

For email field:
- Click ⋮ (three dots) → **Response validation**
- Select **Text** → **Email**

For CGPA field:
- Click ⋮ → **Response validation**
- Select **Regular expression** → **Matches**
- Pattern: `^(\d{1,3}%?|\d{1,2}(\.\d{1,2})?(CGPA|cgpa)?)$`

### Step 5: Customize Theme

1. Click 🎨 **Customize theme**
2. **Header**: Use SoftServe brand colors
3. **Background color**: #f8fafc (light gray)
4. **Font style**: Basic or Modern

### Step 6: Get Shareable Link

1. Click **Send**
2. Click 🔗 **Link** icon
3. ✅ **Shorten URL**
4. Copy link

### Step 7: View Responses

1. Click **Responses** tab
2. See summary charts
3. Click 📊 **Create spreadsheet** to export to Google Sheets
